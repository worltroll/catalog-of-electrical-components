# catalog-of-electrical-components

## Техническое задание
## Назначение
Веб-приложение для просмотра каталога электронных компонентов с возможностью добавления категорий в избранное и управления личным профилем

## Основной функционал
- Главная страница.
- Каталог категорий компонентов с отображением статуса "избранное" для авторизованных пользователей
- Отдельные страницы каждой категории

### Пользователи
- Регистрация
- Авторизация
- Выход из аккаунта

### Личный кабинет
- Редактирование имени, просмотр email, смена пароля, загрузка аватара

### Избранное
- Добавление/удаление категорий в избранное
- Отдельная страница "Любимое" со списком выбранных категорий

## Фичи 
- Кастомные страницы 404 и 400
- Хэшированные пароли
- requirements.txt
- .gitignore

## Технологии
- Bootstrap
- REST API
- Flask
- Flask-Login
- Flask-WTF
- ORM базы данных через sqalchemy
- Храние данных в базе данных
- HTML-шаблоны
- waitress


## Пояснительная записка
### Навигация по файлам
- папка data - 4 .py файла, отвечающих за базу данных
- db/database.db - база данных с пользователями
- папка forms - папка с формами, в ней хранится файл user.py (формы регистрации, автороизации и изменения профиля пользователя)
- папка static - стандартная папка сайта, за исключением папки user в img - папка с изображения профилей пользователей
- папка templates - папка с шаблонами .html файлов
- main.py - основной файл запуска сайта

### Навигация по сайту
- В верхней части сайта располагается его голова - быстрая навигация по страницам.
- В левой её части - страница "О нас" - главная страница проекта
- В правой - работа с профилем, каталог и избранные компоненты
- На вкладке каталог располагаются карточки компонентов, с каждой из которых можно перейти на страницу компонента
- На странице компонента есть небольшое описанике о нём и карусель изображений
